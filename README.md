# סורק קרנות REIT 🏢

כלי שסורק קרנות נדל"ן (REIT) ותעודות סל, שולף נתוני מפתח, שומר היסטוריה ומתריע על תשואת דיבידנד גבוהה.

## מה הוא שולף לכל טיקר
| שדה | מה זה |
|---|---|
| `price` | מחיר נוכחי |
| `div_yield_pct` | תשואת דיבידנד שנתית באחוזים |
| `ffo_per_share` | **FFO למניה** = (רווח נקי + פחת) / מס' מניות. המדד החשוב ל-REIT |
| `p_ffo` | מחיר חלקי FFO — מקביל ל-P/E אבל מתאים לנדל"ן. נמוך = "זול" יחסית |
| `market_cap_b` | שווי שוק במיליארדי דולר |

> **למה FFO ולא EPS?** ב-REIT הפחת החשבונאי על מבנים מנפח כלפי מטה את הרווח הנקי, כך ש-EPS נראה נמוך באופן מטעה. FFO מנטרל את העיוות הזה. ETF-ים (VNQ, XLRE) לא יציגו FFO — הם קרנות סל, לא חברה בודדת.

## הרצה
```powershell
py reit_scanner.py
```

## הגדרות — `config.json`
```json
{
  "tickers": ["O", "VNQ", "XLRE", "PLD", "SPG", "STAG", "AMT"],
  "dividend_yield_alert_pct": 5.0,
  "csv_path": "reit_history.csv"
}
```
- `tickers` — רשימת הטיקרים לסריקה (הוסף/הסר חופשי).
- `dividend_yield_alert_pct` — סף שמעליו תופיע התראה 🔔.
- `csv_path` — קובץ ההיסטוריה (כל הרצה מוסיפה שורות עם חותמת זמן).

## היסטוריה
כל הרצה מוסיפה שורות ל-`reit_history.csv` עם תאריך ושעה — כך אפשר לעקוב אחרי שינויים בתשואה ובמחיר לאורך זמן ולפתוח בקובץ ב-Excel.
